Delivery

Infrastructure

Flujo de Ejecución

1. Entrada

Usuario → canal (Telegram/Web/CLI/Voz) → Hermes Gateway

2. Orquestación

Hermes → JARVIS Core → agente especializado según intención

3. Ejecución

Agente → herramientas MCP → Neo4j/Qdrant/Engram → skills OpenClaw

4. Respuesta

Resultado → verificación → canal original → usuario
