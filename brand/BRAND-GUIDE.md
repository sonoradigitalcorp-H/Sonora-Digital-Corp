# Brand Guide — Sonora Digital Corp / Native Agent OS

## Identity

- **Name**: Sonora Digital Corp (SDC)
- **Product**: Native Agent OS
- **Tone**: Elegant, powerful, warm, precise
- **Voice**: Spanish (MX), direct, clear, no jargon
- **Mascot**: Mystic (the soul of SDC)

## Colors

| Token | Hex | Usage |
|-------|-----|-------|
| `--bg` | `#0a0a0f` | Background |
| `--surface` | `#13131a` | Cards, panels |
| `--border` | `#1e1e2a` | Borders, dividers |
| `--text` | `#e0e0e0` | Primary text |
| `--dim` | `#6b6b80` | Secondary text |
| `--accent` | `#7c5cfc` | Primary accent (purple) |
| `--green` | `#34d399` | Success, online |
| `--yellow` | `#fbbf24` | Warning, degraded |
| `--red` | `#f87171` | Error, offline |
| `--blue` | `#60a5fa` | Info, links |

## Typography

- **Font**: `SF Mono`, `Fira Code`, `Cascadia Code`, monospace
- **Scale**: 0.65rem (tags) → 0.7rem (logs) → 0.8rem (body) → 1.2rem (h1) → 2rem (stats)

## Design Principles

1. **Dark first** — Everything is dark mode. Light mode is not supported.
2. **Terminal aesthetic** — Monospace, minimal, functional. No gradients, no shadows, no rounded corners > 8px.
3. **Status dots** — 🟢 🟡 🔴 for everything (online/warning/offline).
4. **Tags** — Small colored labels for categories, types, status.
5. **Progress bars** — `█` filled, `░` empty, always 10 segments.

## Voice & Tone

| Situation | Tone | Example |
|-----------|------|---------|
| Error | Clear, no blame | "No se pudo conectar con PostgreSQL" |
| Success | Warm, brief | "Todo listo. 122 tools disponibles." |
| Warning | Direct, actionable | "Rate limit excedido. Esperá 60 segundos." |
| Onboarding | Warm, guiding | "Bienvenido. ¿En qué nicho trabajás?" |
| Technical | Precise, minimal | "Gateway OK. 16 capabilities registradas." |
