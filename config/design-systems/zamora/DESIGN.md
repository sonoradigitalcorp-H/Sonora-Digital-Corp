# Zamora — Design System

## Brand Identity
- **Name**: Alejandro Zamora Brand Studio
- **Type**: Personal brand / Creative studio
- **Tone**: Elegant, premium, bold

## Colors
- Primary: #C9A227 (Elegant Gold)
- Secondary: #2D1B69 (Deep Purple)
- Accent: #FF4757 (Crimson)
- Background: #0A0A1A (Deep Black)
- Surface: rgba(201,162,39,0.05)
- Text: #E8E8E8 (Off White)

## Typography
- Display: 'Playfair Display', serif (headings)
- Body: 'Inter', system-ui, sans-serif

## Spacing
- Base: 8px
- Scale: 4, 8, 12, 16, 24, 32, 48, 64, 96

## Components
- Cards: border-radius 16px, gold border on hover
- Buttons: gradient gold-to-crimson, white text
- Borders: rgba(201,162,39,0.2) resting, 0.5 on hover
