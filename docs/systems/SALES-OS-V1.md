## LA OFERTA ÚNICA

Por ahora, **un solo producto**: el producto que ya le vendiste a ABE Music.
Si ABE Music compró X, hay más como ABE Music que compran X.

**Encuentra qué es X. Escríbelo aquí:**

> [Escribe aquí qué compró ABE Music exactamente. ¿Automatización? ¿CRM? ¿Agentes? ¿"Un sistema"?]

**Si no puedes definir X en una oración, no puedes venderlo.**

## PIPELINE (5 etapas)

```
1. LEAD (contacto inicial)
2. CALIFICADO (tiene dolor + presupuesto + urgencia)
3. PROPUESTA (propuesta enviada)
4. NEGOCIACIÓN (objetiones resueltas)
5. CERRADO (contract + pago)
```

## LEAD QUALIFICATION (BANT)

| Criterio | Pregunta |
|----------|----------|
| **B**udget | ¿Tienes presupuesto asignado para esto? |
| **A**uthority | ¿Tú decides o necesitas aprobación? |
| **N**eed | ¿Qué problema específico quieres resolver? |
| **T**imeline | ¿Cuándo necesitas tenerlo funcionando? |

**Hot lead**: Responde sí a 3+ incluyendo Budget + Timeline.
**Warm lead**: 2 respuestas positivas.
**Cold lead**: 0-1 respuesta. No persigas. Pasa a nurture.

## SCRIPTS DE VENTAS

### Primer contacto (LinkedIn DM / WhatsApp)

```
Hola [nombre], vi que [empresa] está haciendo [cosa relevante].
En Sonora Digital Corp ayudamos a [resultado específico] 
a empresas como la tuya.

Ejemplo: [ABE Music] logró [resultado concreto] en [tiempo].

¿Te interesa una llamada de 15 min para ver si podemos ayudarles?
```

### Llamada de descubrimiento (15 min)

```
1. ¿Cuál es tu rol y qué haces? (build rapport)
2. ¿Qué te motivó a tomar esta llamada? (find pain)
3. ¿Qué has intentado antes? (understand context)
4. ¿Qué pasa si no lo resuelves? (urgency)
5. ¿Cómo te gustaría que se viera la solución? (vision)
6. ¿Qué presupuesto tienes en mente? (qualify)
```

### Llamada de cierre (30 min)

```
1. Resumir el problema (muestra que entendiste)
2. Presentar la solución (ALINEADA a su problema)
3. Precio (claro, sin sorpresas)
4. Próximos pasos (contract + pago)
5. Call to action (¿empezamos [fecha]?)
```

## OBJETIVOS SEMANALES

| Métrica | Target |
|---------|--------|
| Leads nuevos contacto | 10/semana |
| Llamadas descubrimiento | 5/semana |
| Propuestas enviadas | 3/semana |
| Clientes nuevos | 1/mes |

**Si no llegas a estos números, no hay pipeline. Si no hay pipeline, no hay revenue.**

## REGLAS DE ORO

1. **El founder vende.** Nadie más. No delegues ventas.
2. **No persigas leads fríos.** Si no responde en 3 seguimientos, pasa a nurture.
3. **Cierra siempre con fecha.** "¿Empezamos el lunes?" > "¿Te interesa?"
4. **Precio primero.** Si no preguntan precio en la primera llamada, no es comprador.
5. **No hagas propuestas sin llamada.** Propuesta sin conversación = archivo.
