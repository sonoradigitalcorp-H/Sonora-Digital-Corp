# Reglas aprendidas del usuario

Cuando el usuario me corrige, anoto aquí para no repetir errores.

## 2026-07-01 — Correcciones de esta sesión

- **Vocabulario**: usar español sencillo, no digas "working tree" ni "remote tracking"
- **Siempre decir en qué branch estoy**: el usuario trabaja en múltiples sesiones paralelas
- **Verificar antes de responder**: no asumir, revisar git status, pwd, etc.
- **Al iniciar sesión**: mostrar resumen automático (carpeta, branch, cambios, otras ramas)
- **No preguntar "qué hicimos"**: leer Engram o session-status.sh automáticamente
- **Ser breve**: máximo 5 líneas al saludar, luego preguntar qué necesita
- **Detectar conflictos entre sesiones**: si otra rama modificó los mismos archivos, avisar
