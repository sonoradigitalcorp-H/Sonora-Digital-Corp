## 1. Objetivo

Agregar capas de inteligencia al Cognitive Kernel: economics (costo por operación), learning (heurísticas desde LECCION.md), agent scoreboard (métricas por agente), y config unification (fleet.yml genera todo).

---

## 2. Value Driver

**founder-independence**: Saber cuánto cuesta cada operación sin hacer cuentas manuales. **automation**: Aprender de errores pasados automáticamente. **knowledge**: Scoreboard permite decidir qué agente usar basado en datos.
