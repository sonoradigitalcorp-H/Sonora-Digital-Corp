## 1. Objetivo
Sistema integrado de respuesta automática en redes sociales con contexto por plataforma, chat inteligente con voz tipo Jarvis, captura de leads vía keyword "Sonora", y agente vendedor IA con conocimiento completo de productos, paquetes y precios.

## 2. Value Driver
Revenue + Founder-independence

## 3. FRs
FR1: Social auto-responder con contexto por plataforma (Instagram, Facebook, TikTok, YouTube)
FR2: Keyword "Sonora" → beneficio freemium + captura lead
FR3: Chat widget con voz (SpeechRecognition + SpeechSynthesis) + IA contextual
FR4: Pop-up de captura de lead con trigger por tiempo y scroll
FR5: Super Seller Agent con conocimiento de productos, paquetes y precios
FR6: Integración con Cyber Diagnosis Express (chat → scan)
FR7: Agente responde en tono de cada plataforma
FR8: Evals + validación de prompts

## 4. Gherkin Scenarios

```gherkin
Feature: Social Auto-Responder
  Scenario: Usuario menciona "Sonora" en Instagram DM
    Given el auto-responder está activo
    When un usuario escribe "Sonora" en un DM
    Then el sistema detecta la keyword
    And responde con link de beneficio freemium
    And captura el lead en state/social/leads.json

  Scenario: Chat widget responde con contexto de productos
    Given el chat widget está abierto
    When el usuario pregunta "¿Cuánto cuesta?"
    Then responde con precios de productos y paquetes
    And ofrece agendar diagnóstico gratis

  Scenario: Voice chat activo
    Given el chat widget está abierto
    When el usuario activa el micrófono
    Then SpeechRecognition captura la voz
    And envía como mensaje de texto
    And el agente responde con voz (SpeechSynthesis)
```

## 5. Technical Approach
- Playwright-based auto-responder (products/social/responder.py)
- Chat widget vanilla JS (frontends/cyber/chat-widget.js)
- SpeechRecognition + SpeechSynthesis para voz
- API Bridge para captura de leads
- Product registry para conocimiento del agente
- Cookies de Chrome para sesiones sociales
