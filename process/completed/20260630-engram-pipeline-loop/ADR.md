# ADR-20260701-001: Engram Pipeline Loop
**Status**: Accepted | **Date**: 2026-07-01
**Context**: El pipeline de procesos no guardaba aprendizaje automáticamente.
**Decision**: Integrar Engram con el pipeline para auto-almacenar lecciones.
**Consequences**: Cada SPEC completada genera automáticamente una lección en Engram.
