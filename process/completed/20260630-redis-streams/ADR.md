# ADR-20260630-004: Redis Streams como Sistema Nervioso
**Status**: Accepted | **Date**: 2026-06-30
**Context**: Los agentes no tenían un sistema de comunicación en tiempo real.
**Decision**: Implementar Redis Streams como sistema nervioso del ecosistema de agentes.
**Consequences**: Comunicación asíncrona, publish/subscribe, persistencia de eventos.
