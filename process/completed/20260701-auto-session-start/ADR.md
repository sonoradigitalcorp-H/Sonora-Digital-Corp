# ADR-20260630-000: Auto Session Start + Session Memory
**Status**: Accepted | **Date**: 2026-07-01
**Context**: Cada sesión empezaba sin contexto, el usuario repetía información.
**Decision**: Implementar auto session start con git status + última sesión + reglas aprendidas.
**Consequences**: El agente sabe dónde está y qué pasó en la sesión anterior.
