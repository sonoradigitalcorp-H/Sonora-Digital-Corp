# AzREC — Roadmap 1 Año

## Mision
Lanzar y consolidar AzREC como marca de musica, gorras y merch desde Hermosillo, Sonora.

## Vision
Ser el sello independiente mas reconocido del noroeste de Mexico, con presencia fisica (casa-estudio-tienda) y digital (Shopify + redes + bots).

---

## Fase 1: FUNDACION (Meses 1-3)
**Objetivo: Infraestructura fisica + digital operando**

### Mes 1 — Setup Inmediato
- [ ] Rentar casa Hermosillo (~20k/mes)
- [ ] Acondicionar cuarto de ensayo (insonorizacion basica)
- [ ] Registrar AzREC como marca (IMPI)
- [ ] Shopify store: gorras + merch (productos iniciales)
- [ ] Landing page live (GitHub Pages)
- [ ] Telegram bot operando con asistente de ventas
- [ ] Canal de YouTube creado
- [ ] Instagram + TikTok business accounts

### Mes 2 — Contenido + Comunidad
- [ ] Primer video YouTube: "Bienvenidos a AzREC"
- [ ] 3 publicaciones/semana en Instagram
- [ ] Grupo Telegram para seguidores
- [ ] Mini-app Telegram: catalogo interactivo
- [ ] Primer pedido Shopify

### Mes 3 — Automatizacion
- [ ] n8n workflow: Instagram → Telegram crosspost
- [ ] Bot interno: AzREC ↔ Hermes comunicación automatica
- [ ] Email marketing list (Brevo/Mailchimp)
- [ ] Analytics dashboard (PostHog)

### Presupuesto Fase 1: ~$35,000 (casa 20k + setup estudio 10k + branding/legal 5k)

**Objetivo: Marca autosuficiente + escalable**

- [ ] Staff: 1 community manager + 1 diseñador
