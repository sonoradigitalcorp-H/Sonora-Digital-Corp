# 1. Fundamentos de Telegram

## Tipos de Entidades
| Tipo | Descripcion | Uso |
|------|-------------|-----|
| **Chat personal** | 1:1 con otro usuario | Soport, ventas directas |
| **Grupo** | Hasta 200K miembros | Comunidad, discusion |
| **Supergrupo** | Grupo con herramientas extra | Comunidad grande, AzREC |
| **Canal** | Broadcast unidireccional | Anuncios, noticias |
| **Bot** | Automatizacion | Ventas, soporte, juegos |

## Diferencias Clave
- **Grupo:** Todos pueden escribir → comunidad
- **Canal:** Solo admins escriben → anuncios
- **Bot:** Automatico → respuestas, ventas
- **Mini-app:** Interfaz web dentro de Telegram → catalogo, juegos

## Estructura Recomendada para Marca
```
Canal @AzREC_Noticias (anuncios)
  ↓
Grupo @AzREC (comunidad, discusion)
  ↓
Bot @AzRECBot (ventas, soporte)
  ↓
Mini-app Catalogo (productos)
```
