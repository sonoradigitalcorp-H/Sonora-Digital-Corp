# Sonora Enterprise OS

Enterprise Transformation Operating System for Sonora Digital Corp.

## Structure

| Directory | Purpose |
|-----------|---------|
| `constitution/` | Supreme governing documents, master prompts |
| `adr/` | Architectural Decision Records |
| `capabilities/` | Business capability registry & maturity model |
| `policies/` | Policy engine rules |
| `events/` | Standard event catalog |
| `skills/` | Skill registry |
| `harnesses/` | Agent harness registry |
| `digital_twins/` | Customer & corporate digital twins |
| `initiatives/` | Initiative registry with kill criteria |
| `metrics/` | Score systems & KPIs |
| `observability/` | Monitoring, FinOps, BI |
| `prompts/` | Prompt evolution engine |
| `playbooks/` | Reusable playbooks |
| `roadmap/` | Autonomous roadmap generator output |
