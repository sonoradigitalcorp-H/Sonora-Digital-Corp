# Skills para Creativos

## fal-ai
Generación de imágenes con fal.ai (FLUX, SDXL).
- Input: prompt, estilo, resolución
- Output: URL de imagen generada
- Uso: `fal-ai generate --prompt "paisaje surrealista" --style "artistic"`

## meme-maker
Crea memes personalizados.
- Input: template, texto superior, texto inferior
- Output: URL del meme
- Uso: `meme-maker --template "drake" --top "Programar a mano" --bottom "Usar IA"`

## video-gen
Generación de videos con IA.
- Input: prompt, duración, estilo
- Output: URL del video
- Uso: `video-gen --prompt "animación de logo" --duration 10 --style "motion"`

## portfolio-builder
Construye portafolios web profesionales.
- Input: proyectos, estilo, secciones
- Output: sitio web de portafolio
- Uso: `portfolio-builder --projects "proyecto1,proyecto2" --style "minimal"`

## brand-identity
Crea identidades de marca completas.
- Input: nombre, valores, audiencia
- Output: logo, colores, tipografía, guía
- Uso: `brand-identity --name "MarcaX" --values "innovación,simplicidad"`
