# Skills para Ecommerce

## shopify
Gestión de tienda Shopify.
- Input: acción (crear producto, actualizar inventario, etc)
- Output: confirmación de acción
- Uso: `shopify create-product --name "Camiseta" --price 299`

## printful
Integración con Printful para print-on-demand.
- Input: diseño, productos
- Output: productos creados en Printful
- Uso: `printful create --design "logo.png" --products "tshirt,mug"`

## fal-ai
Generación de imágenes con fal.ai.
- Input: prompt, estilo
- Output: URL de imagen generada
- Uso: `fal-ai generate --prompt "producto en fondo blanco" --style "professional"`

## analytics
Análisis de métricas de tienda.
- Input: período, métricas
- Output: reporte de analytics
- Uso: `analytics report --period "last-30-days" --metrics "sales,conversion"`

## email-marketing
Automatización de email marketing.
- Input: tipo de campaña, audiencia
- Output: emails programados
- Uso: `email-marketing campaign --type "abandoned-cart" --audience "all"`
