# Skills para Educadores

## course-creator
Crea cursos completos con estructura modular.
- Input: tema, nivel, duración
- Output: curso con módulos, lecciones, evaluaciones
- Uso: `course-creator --topic "Python básico" --level "beginner" --hours 20`

## pdf-generator
Genera PDFs educativos profesionales.
- Input: contenido, estilo
- Output: PDF formateado
- Uso: `pdf-generator --content "material.md" --style "educativo"`

## quiz-builder
Construye quizzes y evaluaciones.
- Input: tema, número de preguntas, dificultad
- Output: quiz con respuestas correctas
- Uso: `quiz-builder --topic "matemáticas" --questions 20 --difficulty "medium"`

## student-tracker
Registra y analiza progreso de alumnos.
- Input: datos del alumno, actividades, calificaciones
- Output: reporte de progreso
- Uso: `student-tracker --student "María" --activity "examen1" --score 85`

## certificate-gen
Genera certificados de finalización.
- Input: nombre alumno, curso, fecha
- Output: PDF de certificado
- Uso: `certificate-gen --student "Juan" --course "Python" --date "2026-06-12"`
