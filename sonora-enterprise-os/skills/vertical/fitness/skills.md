# Skills para Fitness

## content-fitness
Genera contenido visual de fitness con IA.
- Input: tipo de ejercicio, estilo
- Output: imagen/video de ejercicio
- Uso: `content-fitness --type "sentadilla" --style "técnico"`

## voice-tts
Convierte texto a voz para videos de entrenamiento.
- Input: texto, voz (masculina/femenina)
- Output: archivo de audio
- Uso: `voice-tts --text "3 series de 12 reps" --voice "female"`

## progress-tracker
Registra y analiza progreso de clientes.
- Input: medidas, fotos, notas
- Output: reporte de progreso
- Uso: `progress-tracker log --client "Juan" --weight 75 --bodyfat 18`

## nutrition-calculator
Calcula macros y planes nutricionales.
- Input: objetivo, peso, altura, edad
- Output: plan nutricional personalizado
- Uso: `nutrition-calculator --goal "fat-loss" --weight 80 --height 175`

## workout-builder
Construye rutinas personalizadas.
- Input: objetivo, nivel, equipamiento
- Output: rutina completa
- Uso: `workout-builder --goal "strength" --level "intermediate" --equipment "gym"`
