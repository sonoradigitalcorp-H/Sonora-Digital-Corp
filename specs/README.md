# Specs Directory

Active specs are stored in `process/active/` with IDs following `SPEC-YYYYMMDD-NNN`.

Completed specs are archived in `process/completed/YYYYMMDD-<title>/`.

## Current Active Specs

None — all completed specs in `process/completed/`.

## Creating a New Spec

```bash
bash scripts/process-pipeline.sh spec-new "Title"
```

Or use the template at `process/templates/SPEC.md`.
